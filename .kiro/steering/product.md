# Product Overview

AWSomeShop is an internal employee rewards e-commerce platform where employees can redeem products using "AWSome Points".

## Core Purpose
Enable employees to browse and redeem products using a points-based system, while administrators manage product inventory and employee points allocation.

## User Roles
- **Employee**: Browse products, redeem items with points, view order history and points balance
- **Administrator**: Manage products (CRUD), control points allocation, oversee orders
  - SuperAdmin: Full system access
  - ProductAdmin: Product management only
  - PointsAdmin: Points management only

## Key Features
- Product catalog with search, filtering, and categorization
- Shopping cart and order management
- Points-based redemption system with automatic monthly allocation
- Role-based access control (RBAC)
- Background services for scheduled points distribution
